# 第十章与第十一章：文化强国 & 民生社会建设

## 第十章：建设社会主义文化强国

### 【考点1】为什么要建设中国特色社会主义文化
1. 文化是一个国家、一个民族的灵魂
2. 没有文化繁荣兴盛就没有中华民族伟大复兴
3. 更好满足人民日益增长的精神文化需要

### 【考点2】如何提升国家文化软实力和中华文化影响力
1. 推动中华优秀传统文化创造性转化、创新性发展
2. 用中国特色社会主义文化培根铸魂
3. 推进文化事业和文化产业全面发展
4. 讲好中国故事，传播好中国声音

### 【考点3】怎样讲好中国故事
- 着重讲好中国的故事、中国共产党的故事、中国特色社会主义的故事、中国人民的故事，展示文明大国、东方大国、负责任大国、社会主义大国形象，让当代中国形象在世界上不断树立和闪亮起来
- 把握中国故事最精彩的主题：中国共产党为什么"能"、马克思主义为什么"行"、中国特色社会主义为什么"好"

### 【考点4】广泛践行社会主义核心价值观
**基本内容**：
- 国家层面：富强、民主、文明、和谐
- 社会层面：自由、平等、公正、法治
- 公民层面：爱国、敬业、诚信、友善

**重要意义**：
1. 回答了建设什么样的国家、社会和公民的问题
2. 体现社会主义本质要求
3. 继承中华优秀传统文化
4. 吸收世界文明有益成果
5. 有利于巩固社会主义制度、维护社会和谐稳定

**如何践行**：
1. 强化教育引导
2. 注重实践养成
3. 加强制度保障
4. 融入法治建设、社会发展和日常生活
5. 推推进教材、进课堂、进头脑

### 【考点5】传承发展中华优秀传统文化
**地位**：
1. 是中华文明的智慧结晶和精华所在
2. 是中华民族的根和魂
3. 是推动文化自信的重要基础

**中华文明的突出特性**：连续性、创新性、统一性、包容性、和平性

**创造性转化和创新性发展**：
- 创造性转化：根据时代要求，对传统文化中有价值的内容和形式进行改造，赋予新的时代内涵和现代表达
- 创新性发展：根据时代进步，对传统文化内涵进行补充、拓展、完善，增强影响力和感召力

## 第十一章：以保障和改善民生为重点加强社会建设

### 【考点1】经济发展与民生保障的关系
1. 抓民生也是抓发展，做好经济社会发展工作，民生是"指南针"
2. 既要发展经济为持续改善民生奠定坚实物质基础，又要持续不断改善民生为经济发展创造更多有效需求
3. 要全面把握民生和发展相互牵动、互为条件的关系，使民生改善和经济发展有效对接、良性循环、相得益彰

### 【考点2】坚持在发展中增进民生福祉
**核心观点**：发展是解决民生问题的总钥匙，民生是发展的指南针，要在发展中保障和改善民生

**民生和发展的关系**：
1. 发展是改善民生的物质基础
2. 民生改善能够促进发展、扩大内需、增强社会活力
3. 民生和发展相互牵动、互为条件

**工作思路**：坚守底线、突出重点、完善制度、引导预期

**重点任务**：
1. 解决人民群众最关心、最直接、最现实的利益问题
2. 解决群众急难愁盼问题
3. 把为人民排忧解难摆在优先位置
4. 建立常态化、长效化的制度机制

**重要原则**：
1. 尽力而为、量力而行
2. 人人尽责、人人享有
3. 鼓励人民通过辛勤劳动、诚实劳动、创造性劳动创造美好生活

### 【考点3】怎样推进社会治理现代化
1. 坚持系统治理、综合治理、依法治理、源头治理
2. 健全党组织领导的自治、法治、德治相结合的城乡基层治理体系
3. 加强社会治安综合治理

### 【考点4】怎样增强人民获得感、幸福感、安全感
1. 抓住人民最关心最直接最现实的利益问题
2. 坚决打赢脱贫攻坚战
3. 抗疫斗争最大限度保护人民生命安全和身体健康
