## ✨ 特性

- 📱 **移动端适配** - 响应式设计，手机/平板/电脑都能用
- 🔍 **快速导航** - 顶部标签栏，一键跳转任意章节
- 📂 **手风琴折叠** - 点击章节标题展开/收起内容
- ⚠️ **重点高亮** - 黄色提示框突出重要考点
- ↑ **返回顶部** - 滚动超过300px自动显示
- ☰ **汉堡菜单** - 全屏导航，快速定位

## 🚀 使用方式

### 在线访问

直接打开 `index.html` 文件即可在浏览器中查看。

### 本地使用

1. 克隆仓库
   ```bash
   git clone https://github.com/MashedPotato817/2607Xinsixiang-Notes.git
   ```

2. 打开 `index.html` 文件
