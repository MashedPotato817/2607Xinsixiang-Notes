# 第四章与第五章：以人民为中心 & 全面深化改革开放

## 第四章：坚持以人民为中心

### 【考点1】人民立场是中国共产党的根本政治立场
- 体现党的性质宗旨、执政理念和根本属性
- 是中国共产党区别于其他政党的显著标志

### 【考点2】如何理解共同富裕的重要性
1. 共同富裕是中国共产党人始终不渝的奋斗目标
2. 共同富裕是中国特色社会主义的本质要求
3. 共同富裕是中国式现代化的重要特征（五大特征之一）

### 【考点3】扎实推动共同富裕的重大举措
1. 着力扩大中等收入群体规模
2. 加强对高收入的规范和调节
3. 促进人民精神生活共同富裕
4. 促进农民农村共同富裕
5. 促进基本公共服务均等化
6. 提高发展的平衡性、协调性、包容性

### 【考点4】扎实推动共同富裕需要把握的基本原则
1. 坚持基本经济制度
2. 鼓励勤劳创新致富
3. 坚持循序渐进
4. 尽力而为，量力而行

### 【考点5】为什么必须推动全体人民共同富裕取得更为明显的实质性进展
1. 是践行以人民为中心发展思想的必然要求
2. 是更好满足人民日益增长的美好生活需要的重要着力点
3. 是解决我国发展不平衡不充分问题的现实需要
4. 是关系党的执政基础的重大政治问题

### 【考点6】坚持人民立场
1. 坚持以人民为中心的执政理念，将其贯穿于党和国家事业发展各方面
2. 始终牢记党的初心和使命
3. 始终保持党同人民群众的血肉联系
4. 热爱人民、尊重人民、敬畏人民

### 【考点7】经济发展与民生保障的关系
1. 抓民生也是抓发展，做好经济社会发展工作，民生是"指南针"
2. 既要发展经济为持续改善民生奠定坚实物质基础，又要持续不断改善民生为经济发展创造更多有效需求
3. 要全面把握民生和发展相互牵动、互为条件的关系，使民生改善和经济发展有效对接、良性循环、相得益彰

## 第五章：全面深化改革开放

### 【考点1】改革开放的鲜明特征和首要任务
- 解放和发展社会生产力

### 【考点2】全面深化改革开放的主线
- 制度建设

### 【考点3】全面深化改革总目标
1. 完善和发展中国特色社会主义制度
2. 推进国家治理体系和治理能力现代化

### 【考点4】两句话的关系
1. "完善和发展中国特色社会主义制度"规定了改革的根本方向
2. "推进国家治理体系和治理能力现代化"明确了改革的时代要求和鲜明指向
3. 二者是有机统一、不可分割的整体

### 【考点5】"六个紧紧围绕"
1. 围绕发挥市场决定性作用和更好发挥政府作用，深化经济体制改革
2. 围绕坚持党的领导、人民当家作主、依法治国有机统一，深化政治体制改革
3. 围绕建设社会主义核心价值体系、社会主义文化强国，深化文化体制改革
4. 围绕保障和改善民生、促进社会公平正义，深化社会体制改革
5. 围绕建设美丽中国，深化生态文明体制改革
6. 围绕提高科学执政、民主执政、依法执政水平，深化党的建设制度改革

### 【考点6】新发展理念的五大理念及解决的问题
1. **创新**：注重解决发展动力问题
2. **协调**：注重解决发展不平衡问题
3. **绿色**：注重解决人与自然和谐共生问题
4. **开放**：注重解决发展内外联动问题
5. **共享**：注重解决社会公平正义问题

### 【考点7】新发展格局
- 以国内大循环为主体、国内国际双循环相互促进

### 【考点8】为什么要推动高质量发展
1. 推动高质量发展是遵循经济发展规律的必然要求
2. 推动高质量发展是保持经济持续健康发展的必然要求
3. 推动高质量发展是适应我国社会主要矛盾的必然要求

### 【考点9】改革开放是决定当代中国命运的关键一招
1. 进行广泛而深刻的社会变革，推动走出一条中国式现代化道路
2. 解决好生产力和生产关系的矛盾，解放和发展生产力
3. 解决好经济基础和上层建筑的矛盾，完善和发展社会主义制度
4. 吸收人类一切优秀文明成果，创造人类文明新形态
